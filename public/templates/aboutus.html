About Us

Hello visitor!

HnJobs collects a number of job posts published on HackerNews each month, including:

"Ask HN: Who is hiring?"
"Ask HN: Who wants to be hired?"
"Ask HN: Freelancer? Seeking freelancer?"

Data is updated daily.

This website is implemented with the HackerNews API, using MEAN stack (mongoDB, express, Angular, Node). If you think this site looks like being built by an amateur, it is :) I built it as a pet project to learn web development, you can find the source code here.
